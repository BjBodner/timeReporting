# ארכיטקטורת הפרויקט

## קבצים

### index.html
אפליקציית HTML חד-דפית לדיווח שעות עבודה מהבית. כוללת מערכת אימות משתמש מקומית, טיימר, ניהול משמרות, עריכה ומחיקה של דיווחים, סטטיסטיקות, ושמירת נתונים ב-localStorage עם הפרדה לפי משתמש.

### PLAN.md
מסמך תכנון מפורט לאפליקציית דיווח שעות. כולל דרישות, יוזרים, מקרי קצה, שלבי עבודה, ומטריצת עדיפויות.
